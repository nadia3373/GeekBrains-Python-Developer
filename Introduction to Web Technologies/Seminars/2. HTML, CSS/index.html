Почему все любят котиков?

Этому есть много причин (примерно столько же, сколько и самих котиков), но вот некоторые самые важные из них:

  • Они очень милые
  • Они очень мягкие
  • Они очень пушистые
  • Они очень тёплые
  • Они всегда вытворяют что-нибудь этакое
  • Они поднимут настроение в любой ситуации

Виды котиков

Котиков можно классифицировать бесконечно по такому же числу признаков, вот только некоторые из них:

  1. Смешные
  2. Угрюмые
  3. Удивлённые
  4. Ночные
  5. Мейн-куны
  6. Тык
